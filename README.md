## 📋 프로젝트 개요

기상청 날씨 빅데이터 콘테스트에서 **수상한 연구 프로젝트**입니다.

본 연구는 기상 조건을 활용하여 **열수요 예측의 정확도를 향상**시키기 위한 **다중 모델 앙상블 시스템**을 제안합니다.

### 🎯 연구 목표
- 기상 데이터를 활용한 열수요 예측 모델 개발
- 다중 머신러닝 모델의 앙상블 기법 적용
- 예측 정확도 향상 및 실용성 검증

### 🔑 주요 기여
- **다중 모델 앙상블**: 여러 예측 모델을 결합하여 정확도 향상
- **기상 변수 최적화**: 열수요 예측에 효과적인 기상 변수 선정
- **실시간 예측 시스템**: 실무 적용 가능한 예측 파이프라인 구축

## 📊 연구 방법론

### 1. 데이터 수집
- 기상청 기상 데이터 (온도, 습도, 풍속 등)
- 지역별 열수요 데이터
- 시계열 데이터 전처리

### 2. 모델 개발
- **개별 모델**:
  - Extra Trees (ExtraTreesRegressor)
  - CatBoost
  - XGBoost (XGB)
  - LightGBM (LGBM)
  
- **앙상블 기법**:
  - Stacking
  - Weighted Average
  - 모델 성능 기반 동적 가중치 할당

### 3. 평가 지표
- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- R² Score
